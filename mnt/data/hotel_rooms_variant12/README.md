# Практические занятия 9–10. Вариант 12

## Тема

Структуры. Массив структур. Чтение и запись данных в файл.

## Условие задания

Определить структурный тип, описывающий гостиничный номер: название гостиницы, номер, комфортность (`люкс`, `полулюкс`, `стандарт`, `эконом`), количество человек, стоимость, бытовые опции. Заполнить структурный массив 20 записями.

Необходимо:

1. Переписать из исходного массива в другой массив информацию только о тех гостиничных номерах, название гостиницы которых оканчивается на сочетание букв `house`.
2. Новый массив отсортировать по возрастанию количества человек. Алгоритм сортировки — пузырьком.
3. Вывести все данные по конкретному номеру.
4. Вывести 3 самых дорогих люкса и их бытовые опции.
5. Реализовать функцию изменения данных конкретного номера.
6. В отдельный массив поместить все номера, имеющие ванну.
7. Реализовать вывод отфильтрованных данных в виде оберточной функции.
8. Для занятия 10 добавить чтение и запись данных в файл с использованием `fstream`:
   - считать из текстового файла данные числового поля;
   - выполнить запись и чтение массива структур в бинарный файл;
   - оформить чтение и запись в виде подпрограмм.

## Выполненные требования

- Для каждой подзадачи написана отдельная функция.
- Использован перечисляемый тип данных `enum ComfortLevel`.
- В структуре `HotelRoom` есть вложенная структура `Amenities`.
- Ввод данных выполнен инициализацией массива структур внутри программы.
- Сортировка выполнена пузырьком в функции `bubbleSortByCapacity`.
- Использована библиотека `fstream`.
- Чтение и запись текстового и бинарного файлов оформлены отдельными функциями.

## Описание основных структур

```cpp
enum ComfortLevel {
    LUX,
    SEMILUX,
    STANDARD,
    ECONOM
};

struct Amenities {
    bool wifi;
    bool tv;
    bool bath;
    bool balcony;
};

struct HotelRoom {
    char hotelName[60];
    int roomNumber;
    ComfortLevel comfort;
    int capacity;
    double price;
    Amenities amenities;
    bool hasAirConditioner;
};
```

## Основные функции

- `copyRoomsEndingWithHouse` — копирует номера гостиниц, название которых оканчивается на `house`.
- `bubbleSortByCapacity` — сортирует массив пузырьком по количеству человек.
- `printRoomByNumber` — выводит данные по конкретному номеру.
- `printTop3MostExpensiveLux` — выводит 3 самых дорогих номера класса люкс.
- `changeRoomData` — изменяет данные конкретного номера.
- `filterRoomsWithBath` — помещает в отдельный массив номера с ванной.
- `printFiltered` — оберточная функция для вывода отфильтрованных данных.
- `createTextFileForAirConditioner` — создает текстовый файл с числовым полем.
- `readAirConditionerFromTextFile` — считывает числовое поле из текстового файла.
- `writeRoomsToBinaryFile` — записывает массив структур в бинарный файл.
- `readRoomsFromBinaryFile` — читает массив структур из бинарного файла.

## Как запустить

Из корня репозитория:

```bash
g++ src/main.cpp -o hotel_rooms
./hotel_rooms
```

Для Windows в MinGW:

```bash
g++ src\main.cpp -o hotel_rooms.exe
hotel_rooms.exe
```
