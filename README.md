# 📘 Лабораторная работа: Структуры и файлы (C++)

## 📌 Описание проекта

Данный проект объединяет задания **Занятия 9 (Структуры)** и **Занятия 10 (Работа с файлами)**.
Реализована программа на C++, работающая с массивом структур, содержащих информацию о гостиничных номерах.

---

# 📄 Условие задачи (Вариант 12)

Определить структурный тип, описывающий гостиничный номер:

* название гостиницы
* номер
* комфортность (люкс, полулюкс, стандарт, эконом)
* количество человек
* стоимость
* бытовые опции

Заполнить структурный массив 20-ю записями.

Далее выполнить:

1. Переписать в новый массив номера, название гостиницы которых оканчивается на `"house"`.
2. Отсортировать новый массив по возрастанию количества человек (пузырьком).
3. Вывести данные по конкретному номеру.
4. Вывести 3 самых дорогих номера типа "люкс" и их бытовые опции.
5. Реализовать функцию изменения данных номера.
6. В отдельный массив поместить номера с ванной.
7. Реализовать оберточную функцию вывода отфильтрованных данных.

📚 Условие взято из задания 9 (вариант 12)
